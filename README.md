# 대용량 파일 처리 프로젝트

대용량 파일 처리 프로젝트를 진행하며, 배운 내용을 기록하기 위한 템플릿



### 🛠 실습 환경

✔ JDK : AdoptOpenJDK 11

✔ IntelliJ : 2022.2.3
